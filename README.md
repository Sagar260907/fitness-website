# FusionFit - AI Fitness Tracker

A complete fitness tracking website with calorie calculator, meal tracking, exercise library, and AI voice coaching.

## 🌟 Features

- **Calorie Calculator**: Calculate your BMR and daily maintenance calories using the Mifflin-St Jeor formula
- **Meal Tracking**: Log your daily meals with calories, stored persistently in your browser's IndexedDB
- **Exercise Library**: 6 bodybuilding exercises with video demonstrations
- **AI Voice Coach**: Click the voice button on any exercise for detailed audio instructions and form tips
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices

## 🛠️ Technologies Used

- HTML5
- CSS3 with Tailwind CSS
- JavaScript (ES6+)
- IndexedDB for local data persistence
- Web Speech API for AI voice-over
- Font Awesome icons
